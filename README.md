# Journaled

## Description

Journaled is a simple, lightweight note-taking app used to organize notes and ideas as well as editing them.

## Features

- **Note Creation**: Easily create new notes with a simple interface.
- **Note Organization**: Organize notes into categories or tags for easy retrieval.
- **Note Editing**: Edit existing notes with a rich text editor.
- **Search Functionality**: Quickly search for notes using keywords or full-text search.

## Installation
1. clone the repo:
```bash
git clone https://github.com/TiSDi2012/Journaled.git
```

2. run the app using cargo:
```bash
cd Journaled
cargo run
```

## Technology

- **Rust**: my language of choice.
- **Cargo**: Rust's package manager and build tool.
- **egui**: used for the ui.
